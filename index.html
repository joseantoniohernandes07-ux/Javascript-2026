👾 JavaScript 👾

JavaScript es un lenguaje de programación dinámico y de alto nivel para crear interfaces interacticas y dinamicas.

Es uno de los tres pilares fundamnetales del desarrollo web.

  • 1️⃣ HTML: Define la estructura del contenido. Son los elementos que aparecen en la pagina web.
  • 2️⃣ CSS: Define la apariencia, colores, tamaños, estilos, etc.
  • 3️⃣ JS: Define el comportamiento, es decir lo que hace la pagina o sus elementos.

Caracteristiacas de JavaScript

  1. 👉Lenguaje Interpretado: No se necesita compilar. El navegador lo endiende directamente.
  2. 👉Basado en Objetos: Maneja los elementos de HTML como objetos: Window, date, array, class.
  3. 👉Dinámico: Podemos modificar el contenido en tiempo real. (Sin recargar)
  4. 👉Vérsatil y fácil de aprender: Aunque nacio especial mente para la web se usa también para servidores, aplicaciones móviles, videojuegos, animación y por supuesto para IA
